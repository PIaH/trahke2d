# trahke2d
A 2d javascript game that consists out of a mix of thron and snake

## Getting Started

1) Get the content of the folder `public_html`

2) Launch `index.html` in your favorite browser

## Description

The game is created in an html5 canvas element and rendered with javascript.
It supports two users with different input keys:

Player 1) `Arrow Keys (Up, Down, Left, Right)`

Player 2) `WASD  Keys (W, S, A, D)`

## Rules
- Don't hit yourself
- Don't hit some else
- Don't hit the borders of the screen
